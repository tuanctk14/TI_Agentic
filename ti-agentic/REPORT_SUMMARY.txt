REPORT TYPES
================================================================================

Daily Report
  - Period: Single day
  - Size: 6-8 KB, 2-3 pages
  - Create: POST /api/report?type=daily

Weekly Report
  - Period: 7 days
  - Size: 10-12 KB, 3-4 pages
  - Create: POST /api/report?type=weekly

Monthly Report
  - Period: Full month
  - Size: 16-20 KB, 5-8 pages
  - Create: POST /api/report?type=monthly

================================================================================
HOW TO USE
================================================================================

1. CREATE REPORT:
   curl -X POST "http://localhost:8002/api/report?type=monthly"

2. VIEW REPORT LIST:
   curl "http://localhost:8002/api/reports/list"

3. DOWNLOAD REPORT:
   curl "http://localhost:8002/api/report/download?file=monthly_report_2026-05-04_0051.pdf" \
        -o report.pdf

4. FROM UI:
   - Go to Reports tab
   - Click "Create Report"
   - Select type (Daily/Weekly/Monthly)
   - Wait ~2 seconds
   - Download from list
